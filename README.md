# Sales-data-project

Sales & Customer Analytics Case Study
1. Project Overview
This project analyzes retail sales data using Microsoft Excel to uncover insights on revenue, customers, and regional performance. The goal was to transform raw data into an interactive dashboard for decision-making.
2. Objectives
- Analyze overall sales performance
- Identify top-performing products and categories
- Evaluate regional sales
- Understand customer contribution
- Build an interactive dashboard
3. Dataset Description
The dataset contains 500 records with fields such as Order ID, Order Date, Customer ID, Product, Category, Quantity, Price, Revenue, and Region.
4. Data Cleaning
- Converted data into structured table (Sales_Data)
- Standardized date formats
- Removed duplicates
- Recalculated revenue (Quantity × Price)
- Created Month and Year columns
5. Analysis & KPIs
- Total Revenue (SUM)
- Total Orders (COUNTA)
- Total Customers
- Unique Customers (UNIQUE)
- AOV = Revenue / Orders
6. Pivot Analysis
- Revenue over time (monthly trends)
- Revenue by category
- Revenue by region
- Top customers by revenue
7. Dashboard & Interactivity
Slicers added for Category, Region, and Year to allow dynamic filtering across all visuals.

9. Key Insights
- Revenue shows seasonal trends

Excel was effectively used for data cleaning, analysis, and dashboard creation, turning raw data into actionable insights.
